# AnalysisOfApacheHTTP_ServerAccessLog
## Apache HTTP サーバのアクセスログを解析するプログラム

### アクセス件数の集計
Apache のアクセスログファイルを読み込み、アクセス件数を集計する
- 各時間帯毎のアクセス件数
- リモートホスト別のアクセス件数（アクセスの多いリモートホストの順にアクセス件数の一覧を表示する）


### 複数ファイルに対応
解析したいログファイルが複数に分割して保存されている場合にも対応できるよう、入力として複数のログファイルを読み込む

### 期間の指定
期間を絞って集計を行うことができる

### 大規模データへの対応
解析したいアクセスログファイルのサイズが総計で10GB程度、
実行するマシンのメモリが2GBしかない場合でも、メモリ不足にならずに
動作するようにしたい
